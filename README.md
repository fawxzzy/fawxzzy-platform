# Fawxzzy Platform

Durable, public, secret-free contracts for consolidating DiscordOS, Fitness, and Mazer onto one target Supabase platform while preserving product isolation.

This repository does not contain credentials, user data, executable target migrations, Supabase link state, deployment configuration, or permission to mutate any live system. Contract implementation is developed through reviewed branches; migration SQL and live execution require later, separately authorized packets.

Current lifecycle: `BLOCKED` for target writes, data import, Auth import, environment cutover, production deployment, source pause, and source deletion.
